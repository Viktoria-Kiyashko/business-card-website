Этот проект представляет собой сайт-визитку города Ясиноватая. Сайт предназначен для демонстрации информации о городе, его истории, 
инфраструктуре, культурной жизни, а также для взаимодействия с разработчиком через форму обратной связи.
Проект разработан с соблюдением современных стандартов HTML5 и CSS в ходе учебного проекта

## Для запуска проекта на своём компьютере
скачайте данный репозиторий, разархивируйте, 
в папке с файлами кликните джавжды на любой файл формата .html и готово! 
В браузере откроется данный проект


## Использованные технологии
HTML5: Семантические теги для структурирования страниц.
CSS3: Для стилизации элементов, включая адаптивную верстку.
